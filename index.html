Perbedaan Kompresi Citra: Lossy vs Lossless

Tabel Perbedaan

Aspek Lossy Compression Lossless Compression
Definisi Menghilangkan sebagian data citra Tidak menghilangkan data sama sekali
Kualitas Citra Bisa menurun 100% seperti aslinya
Ukuran File Lebih kecil Lebih besar
Reversibilitas Tidak bisa dikembalikan ke bentuk asli Bisa dikembalikan ke bentuk asli
Contoh Format JPEG, WebP (lossy) PNG, TIFF, RAW

Contoh Penggunaan

Lossy Compression

  • Media sosial (Instagram, Facebook)
  • Website untuk loading cepat
  • Aplikasi chatting seperti WhatsApp

Lossless Compression

  • Logo dan desain grafis (PNG)
  • Fotografi profesional (RAW)
  • Dokumen hasil scan (TIFF/PNG)
